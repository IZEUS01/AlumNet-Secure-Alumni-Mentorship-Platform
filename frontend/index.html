Why AlumNet

Built with security from the ground up

Every design decision prioritises your privacy, data integrity, and trust between mentors and students.

✅

Admin-Verified Alumni

Every alumni profile is manually reviewed before they can offer mentorship. No fake accounts, no shortcuts.

🔐

Role-Based Access Control

Students, alumni, and admins each operate in separate, permission-gated environments. Zero privilege leakage.

🛡️

Secure by Design

HTTPS everywhere, bcrypt passwords, CSRF tokens, rate limiting, and immutable security audit logs.

🤝

Structured Mentorship

Send, track, and manage mentorship requests with clear status updates, messages, and alumni response notes.

📎

Document Verification

Alumni upload degree certificates and employment letters. Files are validated server-side for type and size.

📊

Admin Audit Panel

Full visibility into pending verifications, user activity, and platform health — all from a secure admin dashboard.

How AlumNet works

Register

🔍

🔎

Browse & Filter

💬

Request Mentorship

Send a personalised request. Alumni accept or decline with a note. You're connected.
